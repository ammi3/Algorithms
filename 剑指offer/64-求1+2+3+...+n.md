### 64-求1+2+3+...+n

​	题目描述

> 求 `1+2+...+n` ，要求不能使用乘除法、for、while、if、else、switch、case等关键字及条件判断语句（A?B:C）。

​	示例

```java
输入: n = 3
输出: 6
// 2
输入: n = 9
输出: 45
```

​	提示

- `1 <= n <= 10000`

### 1：递归

- 不能用乘法，可以用加法
- 用加法，最直观的用循环，不能用循环的话只能递归
- 递归结束条件：`n=0时，直接返回0`
- 递归的主体：`n=1时，加1；n=2时，加2`。即：`sumNums(n-1)+n`

````java
public int sumNums(int n) {
    if(n == 0) {
        return 0;
    } else {
        return sumNums(n-1)+n;
    }
}
````
